Eazy Board

Spring Boot MVC + Thymeleaf 기반으로 구현한 게시판 서비스입니다. 게시글 작성/조회, 페이징, 회원/작성자 연동을 통해 기본적인 웹 애플리케이션 흐름을 연습했습니다.

게시글 목록 보기 새 글 작성
요약
  • • Spring Boot 2.7, Java 11, Thymeleaf
  • • Spring Security 세션 로그인 + Redis Session
  • • JPA + MariaDB, Soft Delete(`delYn`)

기능 소개

C
게시글 CRUD

목록, 상세, 작성, 수정, 삭제(Soft Delete)를 지원하는 기본 게시판 흐름을 구현했습니다.

P
검색 & 페이징

Spring Data JPA의 Pageable을 사용해 페이징 목록을 구성하고, UX를 고려한 페이지네이션 UI를 적용했습니다.

A
회원/작성자 연동

Author–Post 연관관계를 통해 작성자 정보를 함께 보여주고, 권한(ADMIN/USER)에 따라 화면 요소를 분기합니다.

오늘의 작업

  • DTO 필드명과 템플릿 바인딩 정리
  • 에러 로그(500) 원인(연관관계 누락) 분석 및 해결
